## 로그인 관련

| 증상 | 원인 | 해결 방법 |
| --- | --- | --- |
| 로그인 후 바로 로그아웃됨 | 세션 쿠키 차단 | 브라우저 쿠키 허용, 시크릿 모드 해제 |
| SSO 오류 | IdP 설정 불일치 | 관리자에게 SAML/OIDC 메타데이터 재확인 요청 |
| 계정 잠김 | 로그인 실패 횟수 초과 | 15분 후 재시도 또는 관리자에게 잠금 해제 요청 |

## 성능 관련

| 증상 | 원인 | 해결 방법 |
| --- | --- | --- |
| 페이지 로딩이 느림 | 네트워크 또는 대용량 데이터 | VPN 연결 확인, 필터 범위 축소 |
| 보고서 생성 실패 | 타임아웃 | 조회 기간을 줄이거나 관리자에게 문의 |

## 브라우저 관련

<Steps>
  <Step title="캐시 삭제">
    브라우저 설정에서 캐시 및 쿠키를 삭제한 뒤 다시 로그인합니다.
  </Step>
  <Step title="확장 프로그램 비활성화">
    광고 차단기 등 확장 프로그램이 동작을 방해할 수 있습니다. 비활성화 후 재시도합니다.
  </Step>
  <Step title="지원 브라우저 사용">
    Chrome 또는 Edge 최신 버전을 사용합니다.
  </Step>
</Steps>

## 지원 문의

위 방법으로 해결되지 않는 경우, 아래 정보와 함께 지원팀에 문의해 주세요.

- 발생 일시 및 재현 단계
- 사용 브라우저 및 OS 버전
- 오류 메시지 스크린샷
- 관련 업무 ID (있는 경우)

```text
이메일: support@audigate.com
```
