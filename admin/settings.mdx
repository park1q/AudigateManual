## 일반 설정

| 항목 | 설명 |
| --- | --- |
| **조직명** | 화면에 표시되는 조직 이름 |
| **로고** | 로그인 및 헤더에 표시되는 로고 |
| **시간대** | 보고서 및 로그의 기본 시간대 |
| **언어** | 기본 표시 언어 |

## 알림 설정

- **이메일 알림**: 업무 할당, 마감 임박, 승인 요청 등
- **인앱 알림**: 실시간 알림 센터 표시 여부
- **알림 템플릿**: 이메일 제목·본문 커스터마이징

## SSO / LDAP 연동

<Steps>
  <Step title="IdP 정보 등록">
    SAML 또는 OIDC 제공자(Azure AD, Okta 등)의 메타데이터를 등록합니다.
  </Step>
  <Step title="속성 매핑">
    이메일, 이름, 부서 등 사용자 속성을 Audigate 필드에 매핑합니다.
  </Step>
  <Step title="테스트 로그인">
    테스트 계정으로 SSO 로그인이 정상 동작하는지 확인합니다.
  </Step>
  <Step title="활성화">
    검증 완료 후 SSO를 조직 기본 로그인 방식으로 활성화합니다.
  </Step>
</Steps>

## 보안 정책

- 최소 비밀번호 길이 및 복잡도
- 2단계 인증(2FA) 필수 여부
- 세션 타임아웃 (분 단위)
- IP 허용 목록 (선택)

<Note>
  보안 정책 변경은 모든 사용자에게 즉시 적용될 수 있습니다. 변경 전 영향 범위를 검토하세요.
</Note>
