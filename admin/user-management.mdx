## 사용자 초대

<Steps>
  <Step title="사용자 추가">
    관리 콘솔 → **사용자** → **사용자 초대**를 클릭합니다.
  </Step>
  <Step title="정보 입력">
    이메일, 이름, 부서, 역할(Role)을 입력합니다.
  </Step>
  <Step title="초대 발송">
    **초대 보내기**를 클릭하면 사용자에게 가입 링크가 이메일로 전송됩니다.
  </Step>
</Steps>

## 역할(Role) 관리

| 역할 | 일반적인 권한 |
| --- | --- |
| **Viewer** | 조회만 가능 |
| **Editor** | 업무 생성·수정 |
| **Reviewer** | 검토 및 승인 |
| **Admin** | 전체 관리 권한 |

<Tip>
  조직에 맞게 커스텀 역할을 추가할 수 있습니다. **역할 관리** 메뉴에서 세부 권한을 설정하세요.
</Tip>

## 계정 비활성화

퇴사 또는 권한 회수가 필요한 경우:

1. 해당 사용자를 검색합니다.
2. **비활성화**를 클릭합니다.
3. 비활성화된 계정은 로그인할 수 없으며, 기존 작업 이력은 보존됩니다.

<Warning>
  계정 삭제는 되돌릴 수 없습니다. 일반적으로 **비활성화**를 권장합니다.
</Warning>
