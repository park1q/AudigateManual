## 로그인 및 홈 화면

1. Audigate URL에 접속합니다.
2. 조직에서 발급한 계정으로 로그인합니다.
3. 로그인 후 **홈** 또는 **대시보드**로 이동합니다.

<Info>
  비밀번호를 잊으셨다면 로그인 화면의 **비밀번호 찾기**를 이용하거나 관리자에게 초기화를 요청하세요.
</Info>

## 주요 메뉴

| 메뉴 | 설명 |
| --- | --- |
| **대시보드** | 진행 현황, 지표, 알림 요약 |
| **업무** | 할당된 감사·검토 작업 목록 |
| **보고서** | 생성된 보고서 조회 및 보내기 |
| **설정** | 개인 프로필 및 알림 설정 |

## 기본 작업 흐름

<Steps>
  <Step title="업무 확인">
    **업무** 메뉴에서 본인에게 할당된 항목을 확인합니다.
  </Step>
  <Step title="자료 검토">
    첨부 파일, 증빙 자료를 열어 검토하고 코멘트를 남깁니다.
  </Step>
  <Step title="상태 변경">
    검토 완료 후 상태를 **승인**, **반려**, **보완 요청** 등으로 변경합니다.
  </Step>
  <Step title="보고서 확인">
    필요 시 **보고서** 메뉴에서 결과를 확인하거나 PDF로 보냅니다.
  </Step>
</Steps>

## 유용한 단축 기능

- **검색**: 상단 검색창에서 업무 ID, 제목, 담당자로 빠르게 찾기
- **필터**: 상태, 기간, 담당 부서별 목록 필터링
- **알림**: 우측 상단 벨 아이콘에서 미처리 알림 확인
